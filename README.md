# بوابة الحضور الذكي - Smart Attendance Portal 🏫

نظام متكامل لإدارة الحضور والغياب المدرسي، مصمم ليكون سهل الاستخدام، سريعاً، ويدعم العمل بدون اتصال بالإنترنت (PWA).

## 🚀 المميزات الرئيسية
- **لوحة تحكم المعلم**: واجهة متطورة تعتمد على الكاميرا لتسجيل الحضور بسرعة.
- **بوابة الطالب وولي الأمر**: تتبع حالة الحضور والغياب بشكل مباشر.
- **لوحات تحكم إدارية**: تقارير مفصلة للمديرين ووزارة التربية.
- **دعم PWA**: يمكن تثبيت النظام كتطبيق على الهاتف ويعمل بكفاءة عالية.
- **تصميم عصري**: واجهة مستخدم متميزة باستخدام Tailwind CSS مع دعم كامل للغة العربية (RTL).

## 📂 هيكلية المشروع
- `index.html`: صفحة تسجيل الدخول الرئيسية.
- `dashboard-*.html`: لوحات التحكم الخاصة بالأدوار المختلفة (معلم، مدير، وزارة).
- `portal-*.html`: بوابات الطلاب وأولياء الأمور.
- `scripts/`: تحتوي على المنطق البرمجي وقاعدة البيانات المحلية.
- `styles/`: ملفات التنسيق CSS.
- `assets/`: الصور والأيقونات.

## 🛠 التقنيات المستخدمة
- **HTML5 / JavaScript (ES6+)**
- **Tailwind CSS**: للتنسيق العصري والسريع.
- **Service Workers**: لدعم العمل بدون اتصال (Offline).
- **Google Fonts (Tajawal)**: للخط العربي الأنيق.

## 📥 التثبيت والتشغيل
1. قم بتحميل المشروع (Clone).
2. افتح ملف `index.html` في أي متصفح حديث.
3. لتفعيل ميزات PWA، يفضل تشغيل المشروع عبر سيرفر محلي (مثل Live Server).
